# Monorepo Starter Kit with Turborepo

This repository is a starter kit for creating a monorepo using Turborepo. It includes multiple
packages and applications that can be built and managed together.

## Getting Started

1. Clone the repository:

   ```bash
   git clone
   ```

2. Install dependencies:

   ```bash
   pnpm install
   ```

3. Build the projects:

   ```bash
   pnpm build
   ```

## Packages and Applications

- **apps/web**: A Next.js web application.
- **packages/ui**: A shared UI component library.
